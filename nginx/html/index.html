IoT MQTT Docker Stack

Página mínima servida por Nginx para verificar que el stack IoT industrial está funcionando correctamente.

Servicios del stack

  • MQTT Broker: Mosquitto
  • Base de datos: PostgreSQL
  • Servidor web: Nginx
  • Contenedores: Docker Compose

Este proyecto forma parte de una arquitectura base para prototipos de IoT industrial, Edge AI y monitoreo de datos.
